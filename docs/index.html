Your wish for savings is granted

Ad-Genie finds location-aware deals across seven major retailers and turns any offer into a personalised AI ad poster and a ready-to-post hashtag set — in seconds.

View source on GitHub

What it does

📍

Location-aware deal discovery

Browser geolocation + Google Places find the nearest store for every retailer within 10 miles, surfacing the right offer format automatically.

🎫

Online & in-store redemption

Copy a coupon code at checkout or show a generated QR code in-store — Ad-Genie picks the right format per offer.

🪄

AI ad poster generator

Turns any deal into a personalised, on-brand social poster using OpenAI's image model — with your photo, if you choose.

#

Instant hashtag sets

GPT-4o generates 15–20 ready-to-paste hashtags tuned to the retailer, category, and discount.

📊

Savings dashboard

Twelve KPIs, a 30-day savings chart, and a chronological activity feed — all computed from real redemption data.

🔖

Bookmarking & expiry alerts

Save offers for later and see an expiry countdown before they vanish.
